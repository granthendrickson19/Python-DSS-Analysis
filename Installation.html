Installation¶

For users uninterested in cloning the source directory, they may use the ‘functions’_ simply by using pip to install the package. .. code-block:

''pip install PythonDSSAnalysis'''

With this users can import the package into their current working files with: .. code-block:

''from PythonDSSAnalysis import DSSHandler as DSS''

Users can now call all the ‘functions <functions_>’_ available by invoking the DSS module .. code-block:

''betas = DSS.data_normalizer(someData)''
