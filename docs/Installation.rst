Installation
============
------------
Implementing to Existing Projects
-----------

For users uninterested in cloning the source directory, they may use the 'functions <https://granthendrickson19.github.io/Python-DSS-Analysis/DSSHandler.html#>'_ simply by using pip to install the package.

''pip install PythonDSSAnalysis'''

With this users can import the package into their current working files with:

''from PythonDSSAnalysis import DSSHandler as DSS''

Users can now call all the 'functions <https://granthendrickson19.github.io/Python-DSS-Analysis/DSSHandler.html#>'_ available by invoking the DSS module

''betas = DSS.data_normalizer(someData)''
